## Context

3. **Check documentation files** in `docs/` if the changes affect:
   - Public API surface (new attributes, new properties, changed behavior)
   - Generated code patterns
   - Framework integration behavior
   - Cross-references between doc pages (renamed headings break anchors)

4. **Report findings:**
